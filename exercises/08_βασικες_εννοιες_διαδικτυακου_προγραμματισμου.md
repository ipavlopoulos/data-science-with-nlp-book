# Ασκήσεις κεφαλαίου 8

## Ασκήσεις κατανόησης

1. Τι είναι ένα HTTP request και τι περιλαμβάνει ένα HTTP response;
2. Ποια είναι η διαφορά ανάμεσα σε URL path και query parameters;
3. Γιατί χρειάζεται να κατανοούμε HTML πριν κάνουμε scraping;

## Προγραμματιστικές ασκήσεις

1. Στείλτε GET request σε μια δημόσια σελίδα και εμφανίστε status code, headers και τα πρώτα 500 characters.
2. Δημιουργήστε URL με query parameters και εξηγήστε τι σημαίνει κάθε παράμετρος.
3. Εντοπίστε στο HTML μιας σελίδας τρεις διαφορετικούς τύπους στοιχείων.

## Ερωτήσεις πολλαπλής επιλογής

Σε κάθε ερώτηση μπορεί να υπάρχουν μία ή περισσότερες σωστές απαντήσεις.

1. Τι μπορεί να περιλαμβάνει ένα HTTP response;
   - A. Το όνομα του editor που γράφει τον κώδικα.
   - B. Την προσωπική γνώμη του αναγνώστη.
   - C. Πάντα ένα CSV αρχείο.
   - D. Status code.
2. Ποια είναι query parameters;
   - A. `lang=el` όταν βρίσκεται μετά από `#` fragment.
   - B. `/wiki/Data_science`.
   - C. `&page=2`.
   - D. `?q=data`.
3. Γιατί μας ενδιαφέρει η HTML στο scraping;
   - A. Μας βοηθά να επιλέξουμε στοιχεία.
   - B. Περιέχει tags και attributes.
   - C. Εγγυάται ότι τα δεδομένα είναι ανοιχτά για αναδιανομή.
   - D. Περιγράφει τη δομή της σελίδας.

## Εργασία επέκτασης

- Γράψτε σύντομη τεχνική αναφορά που εξηγεί τη διαδρομή από ένα URL μέχρι την HTML που λαμβάνει ο client.
