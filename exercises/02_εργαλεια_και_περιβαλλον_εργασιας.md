# Ασκήσεις κεφαλαίου 2

## Ασκήσεις κατανόησης

1. Ποια είναι τα πλεονεκτήματα και ποιοι οι κίνδυνοι της χρήσης Jupyter Notebook για ανάλυση δεδομένων;
2. Συγκρίνετε ένα notebook και ένα script ως προς αναπαραγωγιμότητα, τεκμηρίωση και συντήρηση.
3. Τι χαρακτηρίζει ένα “καλό” notebook σε μια εργασία επιστήμης δεδομένων;

## Προγραμματιστικές ασκήσεις

1. Δημιουργήστε ένα μικρό notebook με τρεις ενότητες: φόρτωση δεδομένων, ανάλυση, συμπέρασμα.
2. Γράψτε μια συνάρτηση Python που δέχεται λίστα αριθμών και επιστρέφει πλήθος, μέσο όρο, ελάχιστο και μέγιστο.
3. Πάρτε ένα πρόχειρο/ακατάστατο notebook και αναδιοργανώστε το σε καθαρά markdown και code cells.

## Ερωτήσεις πολλαπλής επιλογής

Σε κάθε ερώτηση μπορεί να υπάρχουν μία ή περισσότερες σωστές απαντήσεις.

1. Ποια χαρακτηριστικά βοηθούν την αναπαραγωγιμότητα ενός notebook;
   - A. Σταθερό seed όπου χρειάζεται.
   - B. Καταγραφή βιβλιοθηκών και εκδόσεων.
   - C. Κελιά που εξαρτώνται από κρυφή κατάσταση.
   - D. Εκτέλεση κελιών με τυχαία σειρά κάθε φορά.
2. Πότε ένα script μπορεί να είναι προτιμότερο από notebook;
   - A. Όταν θέλουμε αφηγηματική παρουσίαση με πολλά ενδιάμεσα σχόλια.
   - B. Όταν θέλουμε επαναλαμβανόμενη εκτέλεση σε pipeline.
   - C. Όταν θέλουμε να κρύψουμε τα ενδιάμεσα βήματα.
   - D. Όταν η ανάλυση εκτελείται μία φορά και δεν θα επαναληφθεί.
3. Ποια στοιχεία χαρακτηρίζουν καλό notebook;
   - A. Καθαρά markdown κελιά.
   - B. Μικρά και εστιασμένα code cells.
   - C. Σαφές τελικό συμπέρασμα.
   - D. Περιορισμένα και σχετικά outputs.
