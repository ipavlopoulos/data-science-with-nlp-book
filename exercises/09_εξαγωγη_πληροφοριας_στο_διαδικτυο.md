# Ασκήσεις κεφαλαίου 9

## Ασκήσεις κατανόησης

1. Γιατί η HTML δεν είναι απευθείας κατάλληλη μορφή για ανάλυση δεδομένων;
2. Τι είναι το DOM και πώς βοηθά στην εξαγωγή πληροφορίας;
3. Ποια είναι η διαφορά ανάμεσα σε αναζήτηση με tag, class και CSS selector;

## Προγραμματιστικές ασκήσεις

1. Χρησιμοποιήστε BeautifulSoup για να εξαγάγετε όλους τους τίτλους μιας σελίδας.
2. Εξαγάγετε links από μια σελίδα και μετατρέψτε τα σχετικά URL σε απόλυτα URL.
3. Δημιουργήστε DataFrame από πληροφορίες που συλλέξατε από HTML στοιχεία.

## Ερωτήσεις πολλαπλής επιλογής

Σε κάθε ερώτηση μπορεί να υπάρχουν μία ή περισσότερες σωστές απαντήσεις.

1. Ποια στοιχεία μπορεί να εξαγάγει ένας scraper;
   - A. Μελλοντικά δεδομένα που δεν υπάρχουν ακόμα στη σελίδα ή σε αίτημα.
   - B. Τίτλους.
   - C. Δεδομένα που δεν υπάρχουν πουθενά στη σελίδα ή σε αίτημα.
   - D. Links.
2. Τι κάνει το `urljoin`;
   - A. Μετατρέπει σχετικό URL σε απόλυτο.
   - B. Συνδυάζει base URL και link.
   - C. Βοηθά όταν τα links είναι σχετικά.
   - D. Βοηθά στη δημιουργία έγκυρων απόλυτων links.
3. Ποια είναι καλά κριτήρια για selectors;
   - A. Να επιλέγουν μόνο τα στοιχεία που θέλουμε.
   - B. Να τεκμηριώνονται.
   - C. Να είναι όσο γίνεται σταθεροί.
   - D. Να βασίζονται σε τυχαία σειρά εμφάνισης χωρίς έλεγχο.

## Εργασία επέκτασης

- Φτιάξτε scraper για μια απλή λίστα σχολίων ή άρθρων και τεκμηριώστε ποια selectors χρησιμοποιήσατε και γιατί.
