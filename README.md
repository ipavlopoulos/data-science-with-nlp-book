# Επιστήμη Δεδομένων με Εφαρμογές στην Επεξεργασία Φυσικής Γλώσσας

Αυτό το αποθετήριο περιέχει τα συνοδευτικά Jupyter notebooks για το βιβλίο *Επιστήμη Δεδομένων με Εφαρμογές στην Επεξεργασία Φυσικής Γλώσσας*.

Τα notebooks ακολουθούν τη δομή των κεφαλαίων και δίνουν εκτελέσιμα παραδείγματα για βασικές έννοιες της επιστήμης δεδομένων, της επεξεργασίας φυσικής γλώσσας και της μηχανικής μάθησης.

## Δομή

Όλα τα notebooks βρίσκονται στον φάκελο [`notebooks/`](notebooks/).

Το κεφάλαιο 12 αφορά την επισημείωση δεδομένων και δεν συνοδεύεται από ξεχωριστό notebook σε αυτή την έκδοση.

## Εκτέλεση

Για τοπική χρήση, κλωνοποιήστε το αποθετήριο και ανοίξτε το με JupyterLab ή Jupyter Notebook:

```bash
git clone https://github.com/ipavlopoulos/data-science-with-nlp-book.git
cd data-science-with-nlp-book
jupyter lab
```

Στη συνέχεια ανοίξτε το notebook που σας ενδιαφέρει από τον φάκελο `notebooks/`.

## Εξαρτήσεις και δεδομένα

Τα notebooks είναι γραμμένα για Python 3. Ορισμένα παραδείγματα χρησιμοποιούν βιβλιοθήκες όπως `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`, `geopandas`, `spacy`, `gensim`, `transformers` ή άλλες εξειδικευμένες βιβλιοθήκες ανά κεφάλαιο.

Μερικά notebooks κατεβάζουν δεδομένα ή εγκαθιστούν προαιρετικά πακέτα κατά την εκτέλεση. Πριν εκτελέσετε ένα notebook από την αρχή μέχρι το τέλος, ελέγξτε τα πρώτα κελιά του και τις εντολές εγκατάστασης που τυχόν περιέχει.
